Target:

DevOps Capstone Project

Description:

Migrating from a monolithic to microservice architecture and implementing a highly scalable DevOps lifecycle.

Concepts Used:

Terraform, Ansible, Git, Kubernetes, Jenkins, AWS.

Tasks:

• Provisioning Infrastructure using terraform.

• Building code triggers in Jenkins when a change is pushed to Git repo. Using webhooks in git.
• Building the docker image and orchestrating the containers using Kubernetes.
