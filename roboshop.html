ROBOSHOP-PROJECT

Target:

Hosting an E-Commerce website online using docker

Description:

Containerizing the complete code of micro-service architectural application of an E- Commerce Roboshop website and hosting it through docker.

Concepts Used:

Dockerfile, Docker compose, Docker volumes, Docker Networking.

Tasks:

• Building images for each of the microservice using Dockerfile.
• Enabling communication between multiple containers by creating a custom bridge network in docker.
• Creating a Volume hosted outside the containers that is independent of containers life cycle.
• Troubleshooting any connectivity issues between containers.
• Exposing the port for web container to public.
